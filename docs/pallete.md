FoodQuest UI Design System (Version 1)
Design Goal
FoodQuest should feel warm, cozy, welcoming, and easy to read. The UI should resemble a charming café or restaurant rather than a futuristic interface. Every screen should use the same colors, typography, spacing, and component styles.

General UI Rules
* Keep layouts simple.
* Avoid clutter.
* Use rounded corners.
* Use consistent spacing.
* Prefer icons with short labels.
* Make every screen follow the same visual language.
The UI should always feel cozy, readable, and polished, matching the atmosphere of a growing restaurant.
